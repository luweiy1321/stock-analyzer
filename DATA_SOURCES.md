## 使用方法

### 1. 查看已有买卖信号（已调整参数）

浏览器打开: **http://localhost:8502**

点击左侧 **"开始批量分析"** 按钮

**调整后的参数**（更容易触发信号）:
- 买入/卖出阈值: **3分**（原来是5分）
- RSI超卖阈值: **35**（原来是30）
- KDJ超卖阈值: **25**（原来是20）

**预期结果**: 24只股票中会有买卖信号

---

### 2. 分析指定股票

在网页应用的输入框中直接输入股票代码，如：
- `AAPL` - 苹果
- `600519.SH` - 贵州茅台
- `300750.SZ` - 宁德时代
- `000001.SZ` - 平安银行
- `MSFT` - 微软
- `NVDA` - 英伟达

---

### 3. 命令行批量分析（推荐）

分析前 30 只股票:
```bash
cd d:\stock-analyzer
python batch_analyzer.py
```

分析指定市场:
```bash
python batch_analyzer.py --market 沪A
python batch_analyzer.py --market 深A
```

保存报告:
```bash
python batch_analyzer.py --save
```
